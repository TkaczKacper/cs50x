World-tag or ZwiftPref

Recommended (for Windows and Mac)
Using the world-tag is a controllable approach using your Zwift preferences file. This method is explained on Zwift Insider. ZwiftPref is a little tool that does the same (and more) made by Jesper Rosenlund Nielsen. It lets you control on which course Zwift starts up. You can find it here.

Do a workout

Recommended (for all platforms)
Nowadays Zwift allows you to select a (public) world outside the schedule when you select a workout. Don't feel like doing a workout? Select/create a free ride workout!

Events, meetups, clubs

Recommended (for all platforms)
By joining an event you can get into event only worlds and routes. Don't want to join an event or don't see an event with your desired world/route? Use the Zwift Companion app to create a meetup or (for even more choice of worlds and routes) create a club and then an event.
